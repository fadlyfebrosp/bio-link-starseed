# Bio Link Starseed

> Live Demo :)

**[Visit Project](https://www.biofadly.my.id/)**

A simple bio link website built using **React** and **Vite**, focused on
speed, simplicity, and modern frontend tooling.

------------------------------------------------------------------------

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open <http://localhost:5173> to view it in your browser.

The page will reload when you make changes.\
You may also see lint errors in the console.

------------------------------------------------------------------------

### `npm run build`

Builds the app for production to the `dist` folder.

It bundles React in production mode and optimizes the build for best
performance.\
The build output is minified and ready to be deployed.

------------------------------------------------------------------------

### `npm run preview`

Runs a local preview of the production build.\
Useful to test the app before deployment.

------------------------------------------------------------------------

### `npm run lint`

Runs ESLint to check code quality and potential issues.

------------------------------------------------------------------------

## Installation

Clone the repository:

``` bash
git clone https://github.com/fadlyfebrosp/bio-link-starseed.git
```

Go to the project directory:

``` bash
cd bio-link-starseed
```

Install dependencies:

``` bash
npm install
```

Start the development server:

``` bash
npm run dev
```

------------------------------------------------------------------------

## React + Vite

This project uses the official **React + Vite** template, providing a
minimal setup with:

-   Fast development server
-   Hot Module Replacement (HMR)
-   ESLint integration

Currently, two official plugins are available:

-   `@vitejs/plugin-react` -- Babel-based Fast Refresh
-   `@vitejs/plugin-react-swc` -- SWC-based Fast Refresh
